SimpleBlog

  • Clone project: git clone https://github.com/sangnguyenplus/simpleblog.git blog
  • cd vào thư mục blog.
  • npm install
  • bower install
  • Tạo db mới tên simpleblog và restore database trong thư mục db.
  • npm start

Website sẽ chạy trên đường dẫn http://localhost:3000.
